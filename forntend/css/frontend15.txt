FontAwsome Icons
=================
If we want to take icons in a CSS then we need to use fontawesome icons.

All fontawesome icons are font based.

To work with fontawesome icons we need to add below CDN link.

ex:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">

ex:1
-----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">		
	</head>
	<body> 
				<i class="fa fa-heart"></i>
				<i class="fa fa-home"></i>
				<i class="fa fa-phone"></i>
	</body>
</html>

CSS Form
=========
Form is used to accept the values from the enduser.

It will send the data to server or database for processing.

ex:

Task3
|
|-----css
	|
	|---mystyles.css
|-----images
	|
	|---micky.png
|
|-----index.html 

index.html
----------
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		
		<link href="https://fonts.googleapis.com/css2?family=Permanent+Marker&display=swap" rel="stylesheet">
		
		<!-- add external CSS -->
		<link rel="stylesheet" href="css/mystyles.css"></link>
		
	</head>
	<body>
			<div class="container">
				<div class="box1">
					<img src="images/girl3.jpg"/>
					<h3>Login Here </h3>
					<form>
						<input type="text" name="t1" autocomplete="off" placeholder="username" required/>
						<br>
						<input type="password" name="t2" autocomplete="off" placeholder="password" required/>
						<br>
						<input type="submit" value="submit"/>
					</form>
				</div>
				<div class="box2">
					<img src="images/micky.png"/>
				</div>
			</div>
	</body>
</html>
