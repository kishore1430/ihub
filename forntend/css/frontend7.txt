2) CANVAS
==========
A <canvas> tag is used to drow the vector graphics via javascript.

A <canvas> tag is a container tag.

A <canvas> tag contains various methods to display lines, circle, rectangle, polygon, graphic images and etc.

ex:1
----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
	</head>
	<body>
			<canvas id="myId" style="border:2px solid black;width:400px;height:400px;">
					
			</canvas>
			
			<script>
				var context=document.getElementById('myId');
				var c=context.getContext("2d");
				c.moveTo(0,0);
				c.lineTo(400,200);
				c.stroke();
			</script>
	</body>
</html>
