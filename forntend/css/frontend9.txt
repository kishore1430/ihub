CSS border property
====================
CSS border property is used to assign the border to an elements.

We have following list of CSS border properties.

1) border-style 

2) border-color 

3) border-width 

4) border shorthand 

1) border-style
-----------------
It is used to set the style to the border.

ex:

<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				border-style:solid;
			}
			h2
			{
				border-style:double;
			}
			h3
			{
				border-style:dashed;
			}
			h4
			{
				border-style:dotted;
			}
			h5
			{
				border-style:inset;
			}
			h6
			{
				border-style:groove;
			}
		</style>
	</head>
	<body>
			<h1>Heading 1 tag </h1>
			<h2>Heading 2 tag </h2>
			<h3>Heading 3 tag </h3>
			<h4>Heading 4 tag </h4>
			<h5>Heading 5 tag </h5>
			<h6>Heading 6 tag </h6>
	</body>
</html>

CSS colors
==========
There are four ways to declare the colors in CSS.

1) Valid color name 

2) Hexa value 

3) RGB value 

4) HSL value 

1) Valid color name 
---------------------
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				color:violet;
			}
			p
			{
				color:forestgreen;
			}
			div
			{
				background-color:tomato;
			}
		</style>
	</head>
	<body>
			<h1> This is Heading Tag </h1>
			<p> This is paragraph Tag </p>
			<div> This is division Tag </div>
	</body>
</html>

CSS margin property
====================
CSS margin is used to provide the space around the elements outside defined border.

We have following list of CSS margin properties.

1) margin-top  

2) margin-right 

3) margin-bottom 

4) margin-left 

5) margin shorthand 

We have following list of margin property values.

ex:
	Name		Description
	------		-------------
	auto		Browser caculate the margin 		 
	value 		Specifies margin in px, pt, cm, em and etc.
	%		specifies a margin in percentage
	inherit 	specifies that the margin should be inherited from the parent element


ex:
----
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				border:2px solid black;
				margin-top:100px;
				margin-right:100px;
				margin-bottom:100px;
				margin-left:100px;
			}
		</style>
	</head>
	<body>
			<h1> This is Heading Tag </h1>

	</body>
</html>

CSS border-radius property 
============================
The border-radius property defines the radius of the element's corners.

We have following list of border-radius properties.

1) border-top-left-radius 
2) border-top-right-radius
3) border-bottom-right-radius
4) border-bottom-left-raidus
5) border-radius shorthand

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				border:2px solid black;
				border-top-left-radius: 5px;
				border-top-right-radius:10px;
				border-bottom-left-radius:15px;
				border-bottom-right-radius:20px;
			}
			
		</style>
	</head>
	<body>	
			<h1> This is Heading Tag </h1>
		
	</body>
</html>


ex:
---
<!DOCTYPE html>
<html>
	<head>
		<title>MyPage!</title>
		<style>
			h1
			{
				border:2px solid black;
				border-radius : 5px 10px 15px 20px;
			}
			
		</style>
	</head>
	<body>	
			<h1> This is Heading Tag </h1>
		
	</body>
</html>

Here border-radius goes to top left, top right, bottom right and bottom left.
