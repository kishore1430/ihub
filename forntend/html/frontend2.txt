Meta tag in HTML
=================
A meta tag is used to declare metadata of a document.

Here metadata means data of a data.

Metadata is used by web browser , search engine and other web resources.

To declare metadata we need to use <meta> tag.

A <meta> tag must declare inside <head> tag.

A <meta> tag is used to declare following things.

1) Description 

2) Keywords 

3) Author 

4) view port 

5) Refresh 

6) Copyright 

and etc.

UTF-8 
======
UTF stands for Unicode Transformation Format.

It is a encoding method which describes what character set a website is written with.


ex:
---
<!DOCTYPE html>
<html>
	<head>
		<!--  title  -->
		<title>MyPage</title>

		<!-- metadata -->
		<meta charset="UTF-8">
		<meta name="description" content="This is html classes by niyaz sir">
		<meta name="keywords" content="Best Training Institute">
		<meta name="author" content="Niyaz Sir">
		<meta name="viewport" content="width=device-width, initial-scale=1">
		<meta http-equiv="refresh" content="05"> 
		<meta name="copyright" content="copy right by niyaz sir">
		
		
	</head>
	<body>
		This is HTML Meta Tag for students
	</body>
</html>

Basic HTML tags
===============

Heading tags
=============
A heading tags are used to display the text in bold and size of text is depends upon number of heading tags.

We have six heading tags i.e h1 to h6.

ex:
---
<!DOCTYPE html>
<html>
	<head>
		<!--  title  -->
		<title>MyPage</title>
		
	</head>
	<body>
		<h1>This is heading1 tag</h1>
		<h2>This is heading2 tag</h2>
		<h3>This is heading3 tag</h3>
		<h4>This is heading4 tag</h4>
		<h5>This is heading5 tag</h5>
		<h6>This is heading6 tag</h6>
	</body>
</html>

Paragraph tag
==============
A <p> tag is used to declare the paragraph.

ex:
<!DOCTYPE html>
<html>
	<head>
		<!--  title  -->
		<title>MyPage</title>
		
	</head>
	<body>
		<p>
			Established in 2000 we have earned a reputation for increasing efficiency, productivity and profitability for our clients through the provision of custom-made software to support, assist with and automate their business activities.
		</p>
	</body>
</html>

</html>
