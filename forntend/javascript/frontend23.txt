Javascript program to Add two textfield data and display the result
====================================================================
<!DOCTYPE html>
<html>
<head>
	<title>MyPage</title>
</head>
<body>
		<table align="center">
			<tr>
				<td>No1:</td>
				<td><input type="text" id="t1"/></td>
			</tr>
			<tr>
				<td>No2:</td>
				<td><input type="text" id="t2"/></td>
			</tr>
			<tr>
				<td>&nbsp;</td>
				<td><button onclick="f1()">ADD</button></td>
			</tr>
			<tr>
				<td>Result:</td>
				<td><p id="result"></p></td>
			</tr>
		</table>

		<script>
			function f1()
			{
				var val1=document.getElementById('t1').value;
				var val2=document.getElementById('t2').value;
				var c=parseInt(val1)+parseInt(val2);
				document.getElementById('result').innerHTML=c;
			}
		</script>
</body>
</html>

document.getElementsByName()
=============================
A document.getElementsByName() returns the group of elements based on the specified names.

ex:
<!DOCTYPE html>
<html>
<head>
	<title>MyPage</title>
</head>
<body>
		Name: <input type="text" name="t1"/>
		<button onclick="f1()"> click </button>

		<div id="result"></div>

		<script>
			function f1()
			{
				var val=document.getElementsByName('t1')[0].value;	
				document.getElementById('result').innerHTML=val;
			}
		</script>	
</body>
</html>

document.getElementsByClassName()
==================================
A document.getElementsByClassName returns group of elements with specified class name .

ex:
<!DOCTYPE html>
<html>
<head>
	<title>MyPage</title>
	<style>
		.myClass
		{
			color:blue;
			text-align: center;
		}
	</style>
</head>
<body>
		<div class="myClass"> This is Div tag </div>

		<button onclick="f1()"> change </button>

		<script>
			function f1()
			{
				var x=document.getElementsByClassName('myClass');
				x[0].innerHTML="This is JavaScript Class";
				x[0].style.backgroundColor="cyan";	
			}
		</script>
</body>
</html>
